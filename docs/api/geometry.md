# geometry

Parametric weld joint geometry via the Gmsh Python API.

- `joints` — `JointGeometry` ABC plus five implementations: `FilletTJoint`, `ButtWeld`, `LapJoint`, `CornerJoint`, `CruciformJoint`.
- `notch` — fictitious 1 mm notch-radius insertion for IIW effective-notch-stress analysis.

::: feaweld.geometry
